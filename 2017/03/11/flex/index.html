Flex

Posted on 2017-03-11 | In CSS

Flex是Flexible Box的缩写，意为”弹性布局”，用来为盒状模型提供最大的灵活性。
任何一个容器都可以指定为Flex布局,包括行内元素。（可以display：inline-flex）
注意Webkit内核的浏览器，必须加上-webkit前缀。（ display: -webkit-flex;）

        .container{
        display: flex;        
        /*主轴方向 row column row-reverse column-reverse*/
        flex-direction: row;

        /*换行 nowrap wrap wrap-reverse*/
        flex-wrap: wrap;

        /*整体：主轴方向上项目的排列 flex-start flex-end center space-between space-around
        当主轴是row时就是说 这几个div都在左边还是右边还是居中等*/
        justify-content: center;

        /*整体：交叉轴方向上项目的排列 flex-start flex-end center space-between space-around 注意要是多条交叉轴才有效 当主轴是row时就是说 这几行都在上面还是下面*/
        align-content: center;

        /*局部：同一主轴 不同交叉轴上的项目对齐方式 flex-start | flex-end | center | baseline | stretch; 当主轴是row时就是说 每一条主轴上的div按照什么对齐方式。上边缘还是下边缘还是居中等 
        如果只有一条交叉轴 相当于align-content*/
        align-items: flex-start;

        height: 600px;
        width: 1000px;
        background-color: pink;
    }
    .box1{
        background-color: black;
        margin: 10px;
        height: 100px;
        width: 100px;
        color:white;
        text-align: center;
        font-size: 18px;
    }
    .box2{
        background-color: white;
        margin: 10px;
        height: 120px;
        width: 140px;
        color:black;
        text-align: center;
        font-size: 18px;
    }
    #b3{
        /*数值越小 越靠前 默认为0*/
        order: -1;

        /*flex-grow属性定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大 如果所有项目的flex-grow属性都为1，则它们将等分剩余空间（如果有的话）。如果一个项目的flex-grow属性为2，其他项目都为1，则前者占据的剩余空间将比其他项多一倍。 也就是说 不会超出固定空间*/
        flex-grow: 1;

        /*flex-shrink属性定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小。 如果所有项目的flex-shrink属性都为1，当空间不足时，都将等比例缩小。如果一个项目的flex-shrink属性为0，其他项目都为1，则空间不足时，前者不缩小。*/
        flex-shrink: 2;

        /*定义了在分配多余空间之前 项目占据的主轴空间 默认auto 就是原本的大小*/
        flex-basis: 100px;

        /*上述三个属性的简写*/
        flex: 1 2 100px;

        /*align-self属性允许单个项目有与其他项目不一样的对齐方式，可覆盖align-items属性。默认值为auto，表示继承父元素的align-items属性，如果没有父元素，则等同于stretch。*/
        align-self: center;
    }
    .czjz{
        margin-top: 20px;
        display: flex;
        flex-direction: row;
        flex-wrap: wrap;
        justify-content: center;
        /*只有一条交叉轴*/
        align-items: center;
        background-color: pink;
        height: 100px;
        width: 100px;

    }
    .son{
        height:60px;
        width: 60px;
        background-color: blue;
    }


<div class="container">
    <div class="box1">1</div>
    <div class="box2">2</div>
    <div class="box1" id="b3">3</div>
    <div class="box2">4</div>
    <div class="box1">5</div>
    <div class="box1">6</div>
    <div class="box2">7</div>
    <div class="box2">8</div>
    <div class="box1">9</div>
</div>
<!-- 构建一个垂直居中的块 -->
<div class="czjz">
    <div class="son"></div>
</div>
